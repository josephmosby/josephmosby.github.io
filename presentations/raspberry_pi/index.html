Arduino code goals

  • Read incoming serial data
  • Turn a light on and off depending on the serial byte received
Arduino code

It's C. Don't freak out.

void setup() {
  Serial.begin(9600);
}

void loop() {
  if (Serial.available() > 0) {
     char inByte = Serial.read();
  }

  if (inByte == '1') {
     digitalWrite(1, HIGH);
  }

  else if (inByte == '0') {
     digitalWrite(1, LOW);
  }
}
Python code, round 1
import serial

ser = serial.Serial('/dev/ttyACM0', 9600)

def turn_on():
  ser.write(b'1')
  return ""

def turn_off():
  ser.write(b'2')
  return ""
