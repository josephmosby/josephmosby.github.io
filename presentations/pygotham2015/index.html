How does a CPU work, anyway?

The Arithmetic Logic Unit!

The Arithmetic Logic Unit!

Okay, so the ALU takes in two binary numbers, and based on the status of certain flags, it computes a function. Got it. I think.

Let's build one!

Inputs: x, y - two 16-bit binary numbers
Control Bits: zx, nx, zy, ny, f, no
Outputs:
x + yx - yy - x01-1
xy-x-y!x!y
x + 1y + 1x - 1y - 1x & yx | y

Outputs: zr if output == 0, ng if output < 0

Our Parts

  • Single-bit OR
  • Single-bit NOT
  • Single-bit AND
  • 16-bit Multiplexer
  • 16-bit bitwise NOT
  • 16-bit bitwise AND
  • A 16-bit incrementer
  • A 16-bit adder

def single_bit_or(a, b):
	'''
	a: single bit, 1 or 0
	b: single bit, 1 or 0
	'''

	if a == 1 or b == 1:
		return 1
	else:
		return 0
					
				
					
def single_bit_not(a):
	'''
	a: single bit, 1 or 0
	'''

	if a == 1:
		return 0
	elif a == 0:
		return 1
					
				
					
def single_bit_and(a, b):
	'''
	a: single bit, 1 or 0
	b: single bit, 1 or 0
	'''

	if a == 1 and b == 1:
		return 1
	else:
		return 0
					
				
					
>>> single_bit_and(1, 0)
>>> 0

>>> single_bit_or(1, 0)
>>> 1

>>> single_bit_not(1)
>>> 0

>>> single_bit_and(1, 1)
>>> 1
